#연간 수혜액 포트폴리오 화면 구성 개선 설명

## 개선 방향

기존 포트폴리오 화면은 복지 정책 카드가 여러 개 나열된 초기 버전에 가까웠습니다.
연간 수혜액 포트폴리오 서비스라면 사용자가 가장 먼저 알아야 하는 것은 다음 네 가지입니다.

1. 내가 받을 수 있는 예상 연간 총액
2. 총액에 실제로 반영된 정책 조합
3. 제외되거나 분리된 정책과 이유
4. 총액을 산정한 기준과 파이프라인

## 화면 구조

```text
상단 히어로
  - 연간 수혜액 포트폴리오
  - 총 예상 현금성 수혜액
  - 즉시 신청/조건 확인 배지

KPI 요약
  - 현금성 예상 총액
  - 추천 조합 수
  - 즉시 신청 가능 건수
  - 확인 필요 건수
  - 분리/제외 건수

금액 구성
  - 카테고리별 연간 수혜액
  - 중복/조건 제외 건수

추천 연간 수혜 조합
  - 정책명
  - 연간 반영 금액
  - 신청 상태
  - 산정 근거
  - 총액 반영 여부
  - 조건 적합 이유
  - 다음 행동
  - 원문 근거

제외/분리 판단
  - 대출형 정책
  - 중복 불가 정책
  - 조건 불일치 정책

산정 파이프라인
  - 사용자 조건 입력
  - 정책 후보 점수화
  - 연간 수혜액 정규화
  - 원문 근거 기반 추론
  - 중복/제외 필터링
  - 연간 수혜액 최대 조합 선택
```
